- type deduction
Reference Link: https://en.cppreference.com/w/cpp/language/template_argument_deduction
functiont template
-----------------------
template<typename T>
void f(ParamType param);

f(expr);    // calling function 'f'
-----------------------
two types of type deduction
1. about T
2. about ParamType

Three cases
1. ParamType is a pointer or a reference but not an universal reference.
2. ParamType is an universal reference.
3. ParamType is not both a pointer and a reference.
